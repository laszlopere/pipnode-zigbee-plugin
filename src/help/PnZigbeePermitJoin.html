Zigbee Permit Join

On any incoming message, emits a Zigbee2MQTT bridge command that opens the pairing (permit-join) window for a preset interval.

The outgoing message targets zigbee2mqtt/bridge/request/permit_join with a { "value": true, "time": <seconds> } payload, ready for a downstream MQTT Sink. The window length is configured in minutes (default 4) and converted to seconds (minutes × 60).
